# Guia para os alunos

## Como usar

-   `http://localhost:80` - Link de acesso.
-   `Dashboard lateral esquerda` - Escolher desse menu o desafio que queira responder.
-   `Escolher Arquivo` - Botão para selecionar o código fonte com sua solução.
-   `Enviar` - Botão de envio.
